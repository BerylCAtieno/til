1. docker images: List all images available on the local system.
    
    docker images

2. docker pull: Pull an image or a repository from a registry.

    docker pull <image_name>:<tag>

3. docker build: Build an image from a Dockerfile.

    docker build -t <image_name>:<tag> <path_to_Dockerfile>

4. docker rmi: Remove one or more images from the local system.

    docker rmi <image_id>

5. docker tag: Tag an image into a repository.

    docker tag <source_image>:<tag> <target_repository>:<tag>

6. docker history: Show the history of an image.

    docker history <image_name>:<tag>

7. docker inspect: Display detailed information on one or more images.

    docker inspect <image_name>

8. docker save: Save one or more images to a tar archive.

    docker save -o <output_file.tar> <image_name>:<tag>

9. docker load: Load an image from a tar archive.

    docker load -i <input_file.tar>

10. docker prune: Remove all unused images.

    docker image prune

11. docker search: Search for an image on a Docker registry.

    docker search <search_term>

12. docker push: Push an image or a repository to a registry.

    docker push <image_name>:<tag>
